## What The Product Is

Aliyun ISI is Alibaba Cloud's Intelligent Speech Interaction product. It covers speech recognition, speech synthesis, long-text synthesis, file transcription, and related speech interaction scenarios.

## Practical Setup Sequence

1. Activate the service.
2. Ensure the acting account or RAM user has the needed permissions.
3. Create or select a project in the ISI console.
4. Retrieve the `AppKey`.
5. Use AccessKey credentials on the server side to obtain or manage short-lived auth for clients.
6. Integrate the chosen SDK or API.
